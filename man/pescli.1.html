pescli(1) General Commands Manual pescli(1)

NAME

pescli [OPTION]... [FILE]

DESCRIPTION

Execute Pesc instructions from FILE and display the result. If no file is given, read from stdin.

pescli uses the Pesc calculation language; see pesc(7) for more information.

OPTIONS

-h, --help
Print a short help message and exit.

-V, --version

Print pescli's version and exit.

-q, --quiet

Print as little information as possible.
